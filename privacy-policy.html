一、引言

欢迎使用「洛克王国助手」（以下简称"本应用"）。我们非常重视您的隐私保护。本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的个人信息。

请您在使用本应用前仔细阅读本隐私政策。使用本应用即表示您同意本隐私政策的全部内容。

二、应用信息

应用名称：洛克王国助手

开发者：Misaka-666

应用版本：2.0.2

应用类型：游戏辅助工具

三、我们收集的信息

为了提供更好的服务，本应用可能收集以下信息：

1. 网络访问权限

  • 权限名称：ohos.permission.INTERNET
  • 用途：用于获取游戏商人商品数据、精灵图鉴数据等公开信息
  • 数据来源：rocokingdomworld.org 等公开数据源

2. 通知权限

  • 权限名称：ohos.permission.PUBLISH_AGENT_REMINDER
  • 用途：用于在商品刷新时发送本地提醒通知
  • 说明：仅用于本地通知，不会上传任何数据

3. 设备信息

  • 收集内容：设备型号、系统版本、屏幕分辨率
  • 用途：用于适配不同设备，优化显示效果

四、信息使用目的

我们收集的信息将用于以下目的：

  • 提供远行商人商品查询功能
  • 提供孵蛋鉴定功能
  • 提供精灵图鉴查询功能
  • 发送商品刷新提醒通知
  • 优化应用性能和用户体验
  • 修复已知问题和漏洞

五、信息存储与保护

1. 数据存储

  • 本应用主要采用本地存储方式
  • 精灵图鉴数据内嵌于应用中
  • 孵蛋鉴定数据通过 GitHub 自动更新
  • 远行商人数据实时从服务器获取，不本地持久化存储

2. 数据安全

  • 所有网络请求均使用 HTTPS 加密传输
  • 不收集、不存储用户的个人身份信息
  • 不与任何第三方共享用户数据

六、第三方服务

本应用使用以下第三方服务：

rocokingdomworld.org

提供洛克王国游戏相关数据，包括商人商品、精灵信息等公开数据。

GitHub

用于托管和自动更新孵蛋鉴定数据、精灵图鉴详情数据。

这些第三方服务有各自的隐私政策，我们建议您查阅其相关政策。

七、用户权利

您享有以下权利：

  • 知情权：了解我们如何收集和使用您的信息
  • 访问权：随时查看本隐私政策

  • 删除权：卸载应用将删除所有本地数据
